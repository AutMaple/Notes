# IDEA 常用快捷键

代码编辑相关的快捷键

| 快捷键          | 作用                                                         |
| --------------- | ------------------------------------------------------------ |
| Alt + Enter     | 提示修改错误的方法                                           |
| Alt + Insert    | 插入构造方法，getter, setter等                               |
| Alt + Shift + k | 自动补全方法的返回值和类型。也可以使用（new User().var 然后按下 Tab 键，IDEA 也会补全类型 User user） |
| Ctrl + O        | 填充父类和接口的方法                                         |
| Ctrl + Alt + m  | 将选中的区域变成一个方法                                     |

查看代码相关的快捷键

| 快捷键     | 作用                         |
| ---------- | ---------------------------- |
| Ctrl + h   | 显示类或接口的继承链和实现类 |
| Ctrl + F12 | 显示类的所有方法和属性       |
